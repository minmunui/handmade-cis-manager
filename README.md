# Handmade CIS Manager

연구실 인원과 일정을 관리하기 위한 프로젝트입니다.

Notion의 Database 기능을 인터페이스로 사용하여 Discord Bot을 통해 연구실 인원과 일정을 관리할 수 있습니다.

**이 프로젝트는 개발 공부를 위해서 생성형 AI Agent를 사용하지 않았습니다.**

> ### Background
>
> 지금까지의 모든 일정 관리는 관리자가 직접 이메일/카톡을 발송하는 형태로 작성되었으며, 각 프로젝트는 산개되어 있었습니다.
> 
> 이는 간단한 프로그램으로 해결할 수 있으나, 연구실의 일정 하나만을 위해 특정 웹 사이트에 접속하거나 앱을 설치하는 것은 번거롭습니다.
> 
> 또한, 개발 인력이 부족한 장난감 프로젝트의 특성상, 낮은 완성도와 지속적인 유지보수가 어렵다는 문제점이 발생합니다.
> 
> 때문에, 이미 잘 만들어져 있는 플랫폼에 자동화 기능을 추가하여, 우리가 사용하는 플랫폼 내에서 모든 일정 관리를 해결하고자 합니다.
>
> Notion의 Database 기능을 UI로 사용하고, Discord Bot을 통해 채팅과 알림, 권한 관리를 수행하는 시스템을 구축합니다.


## 주요 기능

### Notion Database

- 연구실 인원 관리
    - 이름, 역할, 연락처 등 정보 저장
    - Email 전송을 통한 Discord 초대장 발송
    - Discord Bot을 통한 회원 검증
    - 그룹을 통한 분류 및 관리 + 그룹에 따른 Discord Channel/Category 자동 생성 및 권한 부여
    - Admin, Member, Guest 권한 분리
    - Notion Database에 따라 권한 자동 부여 및 회수
- 연구실 일정 관리
    - Discord Bot을 통한 관련 인물 Mention을 통한 일정 추가 알림 및 리마인드
